# JAK SKONFIGUROWAĆ TEN VAULT W OBSIDIANIE

Plik instruktażowy — przeczytaj zanim otworzysz vault.

---

## 1. Instalacja Obsidiana

Pobierz z: https://obsidian.md (bezpłatny)

---

## 2. Otwieranie vaultu

1. Uruchom Obsidiana
2. Kliknij **"Open folder as vault"**
3. Wskaż folder **jung-vault**
4. Vault się otworzy — zacznij od `🧠 MÓZG JUNGA — START.md`

---

## 3. Widok Grafu (najważniejszy!)

Naciśnij `Ctrl+G` (lub `Cmd+G` na Mac) → zobaczysz **całą sieć połączeń**.

### Zalecane ustawienia grafu:
- **Node size**: według liczby połączeń (Links)
- **Forces**: zmniejsz odpychanie, zwiększ przyciąganie centralne
- **Groups**: ustaw kolory według folderów

---

## 4. Nawigacja

| Skrót | Akcja |
|---|---|
| `Ctrl+G` | Graf globalny |
| `Ctrl+E` | Przełącz edycja/podgląd |
| `Ctrl+P` | Szybkie otwieranie (znajdź notatkę) |
| `Ctrl+Shift+F` | Wyszukaj we wszystkich notatkach |
| Klik na `[[link]]` | Otwórz połączoną notatkę |
| `Alt+←` | Wstecz (historia nawigacji) |

---

## 5. Zalecane wtyczki (Community Plugins)

Włącz w: Settings → Community plugins → Browse

| Wtyczka | Zastosowanie |
|---|---|
| **Dataview** | Dynamiczne listy notatek według tagów |
| **Graph Analysis** | Analiza struktury grafu |
| **Kanban** | Tablica do śledzenia czytania |
| **Natural Language Dates** | Daty w języku naturalnym |

---

## 6. Tagi — szybkie filtrowanie

Każda notatka ma tagi. Kliknij w panelu tagów (prawy panel) aby filtrować:

- `#archetyp` — wszystkie archetypy
- `#indywiduacja` — droga ku Jaźni
- `#sny` — praca ze snami
- `#alchemia` — symbolika alchemiczna
- `#terapia` — aspekty kliniczne
- `#synchroniczność` — zbiegi okoliczności
- `#fundament` — podstawowe koncepty

## 8. CSS Snippet (opcjonalnie — kolorowy graf)

1. Skopiuj plik `assets/jung-graph.css` do folderu `.obsidian/snippets/`
2. Idź: Settings → Appearance → CSS Snippets
3. Włącz `jung-graph`
4. Graf pokaże różne kolory dla różnych obszarów
